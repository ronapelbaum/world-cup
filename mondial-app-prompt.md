# פרומפט ל-Claude Code: אפליקציית "המדריך הגדול למונדיאל 2026" ⚽

> העתק את כל המסמך הזה כהודעה ל-Claude Code (או כלי סוכן דומה) בתיקייה ריקה.

---

## 0. מה לבנות (תקציר מנהלים)

אפליקציית ווב **חינוכית וכיפית לילד בן 10** שמלמדת על מונדיאל 2026 (ארה"ב/קנדה/מקסיקו, 48 נבחרות).
האפליקציה ב**עברית, RTL מלא**, עם עיצוב **צבעוני ומשחקי בסגנון אלבום מדבקות**.

עקרון מפתח: **הפרדה מוחלטת בין מידע (קבצי JSON) לבין הרינדור (קומפוננטות React)**. כל הוספת נבחרת = הוספת קובץ JSON אחד, בלי לגעת בקוד. היעד הסופי הוא 48 נבחרות; הפרומפט הזה מספק 13 נבחרות התחלתיות + סכמה + הוראות להשלמה.

**Stack:** React + Vite (JavaScript, לא TypeScript אלא אם תעדיף). ללא ספריות UI כבדות. CSS רגיל עם CSS variables. אנימציות ב-CSS, צלילים ב-Web Audio API, אין תלות חיצונית מיותרת.

## 3. מערכת העיצוב (`tokens.css` — מקור אמת יחיד)

**Vibe:** אלבום מדבקות כדורגל — צבעוני, עליז, מסגרות לבנות עבות, פינות מעוגלות, צללים "מודבקים" (offset), אנימציות קופצניות. בהיר ושמח, מתאים לנוער. RTL מלא.

**פונטים:** Rubik (כותרות, משקל 900) + Heebo (גוף) מ-Google Fonts — שניהם תומכים עברית מצוין. אל תשתמש ב-Arial/Inter/system fonts.

```css
:root{
  --grass:#2bb24c; --grass-dark:#159a3a;
  --sky-1:#36bdf6; --sky-2:#1f8fe0;
  --sun:#ffd23f;   --sun-deep:#ffb800;
  --coral:#ff5d5d; --ink:#13233b; --ink-soft:#3a5172;
  --shadow:0 10px 0 rgba(19,35,59,.12), 0 18px 30px rgba(19,35,59,.18);
  --shadow-sm:0 4px 0 rgba(19,35,59,.12), 0 8px 16px rgba(19,35,59,.14);
  --radius:24px;
}
/* רקע גוף: שכבות גרדיאנט שמש+שמיים+דשא */
body{ background:
  radial-gradient(circle at 12% 8%, rgba(255,210,63,.55), transparent 38%),
  radial-gradient(circle at 88% 0%, rgba(255,93,93,.40), transparent 40%),
  linear-gradient(180deg,#7fd3ff 0%,#aee9ff 30%,#d7f6e1 70%,#c7f0d2 100%);
  background-attachment:fixed; }
```

**צבעי נבחרת דינמיים:** כל baner/כרטיס/מודאל צובע את עצמו מ-`team.colors` (`linear-gradient(135deg, c2, c1)`). תווית התפקיד צבועה לפי `positions[line].color`.

**כללי עיצוב:** מסגרות לבנות 3–5px, `--shadow` למרכיבים גדולים ו-`--shadow-sm` לקטנים, hover עם `translateY(-3px)` קל וסיבוב זעיר. כותרות סקשן עם "קו מקווקו" צבעוני. כדורי רקע צפים (⚽🏆) ב-opacity נמוך.

## 4. מסכים ותכונות

### 4.1 ניווט משולב (חובה)
- **טאבים עליונים לפי נושא:** ⚽ הנבחרות · 📊 איך זה עובד · 📅 קו הזמן · 🗺️ מפת האצטדיונים.
- בתוך "הנבחרות": **טאבי נבחרת** (גלולות צבעוניות עם דגל) שמחליפים את הנבחרת המוצגת. הטאב הפעיל צבוע ב-`c2` של הנבחרת.

### 4.2 מסך נבחרת
1. **Hero:** דגל ענק, שם, **כינוי ישראלי**, ו-funstat.
2. **כרטיסי כוכבים:** רק שחקנים עם `star:true`. כל כרטיס: אווטאר/תמונה, שם, תווית תפקיד מדויק, באדג' ⭐. **לחיצה פותחת מודאל** (לא קלף מתהפך).
3. **סגל מלא:** מקובץ לפי קו (שוערים→הגנה→קישור→התקפה), כל שחקן כ"צ'יפ" עם מספר + שם + תפקיד מדויק. **לחיצה על כל שם פותחת את אותו מודאל.**
4. **מודאל שחקן (`PlayerModal`):** חלון מרווח (מונע חיתוך טקסט) עם: תמונה/אווטאר גדול, שם+דגל, תווית תפקיד, רשת 3 נתונים (מספר/גיל/קו), קופסת מועדון, קופסת "💡 עובדה מגניבה", וקופסת "🤩 הידעת?" (אם קיים). סגירה ב-✕, לחיצה ברקע, ו-Esc.
5. **חידון נבחרת:** שאלה-שאלה עם ניקוד, צבע ירוק לתשובה נכונה / אדום+רעד לשגויה, מסך סיכום, ו"שחק שוב". ניקוד מלא → צליל ניצחון + קונפטי.

### 4.3 "איך זה עובד" — כולל 2 תרשימים חדשים
- **`GroupStageDiagram` (תרשים שלב הבתים):** הצג סכמטית בית לדוגמה (4 נבחרות), חיצים ל"2 ראשונות עולות" + הסבר "8 שלישיות הטובות", ובסך הכל 48 → 32. ויזואלי, צבעוני, לא טבלה יבשה.
- **`PointsCalculator`:** כפתורי ניצחון(3)/תיקו(1)/הפסד(0) שמעדכנים מספר נקודות גדול + צליל + קונפטי בניצחון. (קרא מ-`structure.points`.)
- **`KnockoutBracket` (תרשים הנוקאאוט):** עץ מסלול: 32 → 16 → 8 → 4 → גמר 🏆, אינטראקטיבי (hover/לחיצה מדגישה שלב). הסבר קצר: "מפסיד יוצא הביתה", הארכה, פנדלים.

### 4.4 קו הזמן (`Timeline`)
תחנות מ-`schedule.json`, כל תחנה עם נקודה צבעונית+אימוג'י; **לחיצה פותחת/סוגרת** את התיאור (אקורדיון). 

### 4.5 מפת אצטדיונים (`StadiumGlobe`) — חדש
**גלובוס סכמטי ופשוט** (SVG): עיגול כדור-הארץ עם רמז ליבשת צפון/מרכז אמריקה, ו-16 סיכות (נקודות זוהרות) במיקומי הערים מ-`stadiums.json`. השתמש בהיטל פשוט (lng→x, lat→y, ליניארי על אזור צפון-אמריקה — לא צריך היטל גאוגרפי מדויק). **לחיצה/hover על סיכה** מציגה כרטיסון: עיר, מדינה (דגל), אצטדיון. צבע סיכות לפי מדינה (🇺🇸/🇨🇦/🇲🇽). שמור על "סכמטי ופשוט" — לא מפה אמיתית, לא ספריית מפות.

### 4.6 צלילים ואנימציות (`lib/`)
- `sound.js` — Web Audio (ללא קבצים): `click, open, tab, right, wrong, win`. אתחל AudioContext רק במחווה ראשונה של המשתמש.
- `confetti.js` — פיצוץ קונפטי בניצחון חידון / ניצחון מחשבון.
- אנימציות כניסה (stagger) לכרטיסים, hover קופצני, מודאל עם popin.

### 4.7 אווטארים ותמונות (`lib/avatar.jsx`) — חשוב לזכויות יוצרים
- ברירת מחדל: **אווטאר מאוייר** (SVG) שנבנה מצבעי הנבחרת + מספר החולצה + ⭐ לכוכב. **אין להשתמש בתמונות אמיתיות של שחקנים** (תמונות סלבריטאים מוגנות בזכויות יוצרים).
- אם שדה `img` של שחקן מכיל URL — הצג אותו במקום האווטאר (חתוך בעיגול). זה מאפשר למשתמש להוסיף בעצמו תמונות עם רישיון חופשי (למשל מוויקימדיה). אל תאחזר/תוריד תמונות באופן אוטומטי.

## 5. כללי תוכן

1. **שפה:** עברית, גוף פונה לילד בן 10 — חם, פשוט, מסביר מונחים. RTL מלא.
2. **כינויי נבחרות — המקובלים בישראל בלבד.** אסור להמציא כינוי. אם אינך בטוח לגבי נבחרת — השאר `nick` ריק והוסף הערת `// TODO: לאמת כינוי ישראלי`. כינויים מאומתים שמותר להשתמש בהם:
   - ארגנטינה → **האלביסלסטה** · ברזיל → **הסלסאו** · צרפת → **הטריקולור**
   - אנגליה → **שלושת האריות** · ספרד → **לה רוחה** · גרמניה → **המאנשאפט**
   - הולנד → **האורנייה** · איטליה → **האדזורי** · בלגיה → **השדים האדומים** · מרוקו → **אריות האטלס**
   - פורטוגל / ארה"ב / אורוגוואי → השאר ריק עד אימות (אל תמציא).
3. **דיוק:** שדות גיל/מועדון/מספר עשויים להשתנות — היכן שלא מאומת, השתמש ב-`null`/`""` והוסף הערה. אל תמציא עובדות.
4. **בטיחות:** תוכן ידידותי ומתאים לגיל בלבד.
